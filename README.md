# 🏏 IPL Data Analysis (2008–2026)

An Exploratory Data Analysis (EDA) of the Indian Premier League (IPL) 
spanning 18 seasons — uncovering trends in team performance, player 
statistics, match outcomes, and more.

---

## 📌 Project Overview

This project analyzes IPL match data from 2008 to 2026 using Python. 
The goal is to extract meaningful insights from historical data through 
data cleaning, visualization, and statistical analysis.

---

## 🔍 Key Analysis Performed

- 🏆 Most successful teams across all IPL seasons
- 🏏 Top run-scorers and highest strike rates
- 🎯 Top wicket-takers and economy rates
- 🪙 Toss decision impact on match results
- 🏟️ Venue-wise win statistics
- 📈 Season-wise trends in scores and results
- 🥇 Player of the Match award analysis
